# Content Management Guide

The website content is managed through the Decap CMS admin panel — no coding or JSON editing required.

## Logging In

1. Go to [https://hbbawater.com/admin](https://hbbawater.com/admin)
2. Click **Login with Netlify Identity**
3. Use your invited email and password

> If you haven't been invited yet, ask a developer to send you an invitation from the Netlify dashboard.

---

## What You Can Edit

| Section | What it controls |
|---|---|
| **Site Settings** | Site title, hero section, about blurbs, contact info, CCR banner, news notices, quick links, and SEO metadata |
| **Alerts** | Homepage announcements and service notices |
| **Documents** | Downloadable files (meeting minutes, reports, forms) |
| **Governance** | Board members, employees, consultants, and memberships |
| **Helpful Links** | External links grouped by category |
| **Gallery** | Slideshow images on the homepage |
| **Rates & Charges** | Water rates, fees, and footnotes |

---

## Making Changes

1. Log in to the admin panel
2. Select the section you want to edit from the left sidebar
3. Make your changes using the form fields
4. Click **Save** — changes are committed to the repository automatically
5. Netlify will detect the commit and redeploy the site (usually takes 1–2 minutes)

---

## Getting Help

If you can't log in or have questions, contact a developer.
